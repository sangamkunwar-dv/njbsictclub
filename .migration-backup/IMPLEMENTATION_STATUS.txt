================================================================================
                       SUPPORT & TROUBLESHOOTING
================================================================================

If contact form doesn't work:
  1. Check MONGODB_URI is set correctly
  2. Verify MongoDB cluster is running
  3. Check browser console for errors
  4. See TROUBLESHOOTING.md for details

If emails don't send:
  1. Verify email API key (Resend or SendGrid)
  2. For Resend: check email is verified
  3. For SendGrid: check sender email is verified
  4. Check spam folder in inbox
  5. Review browser console for errors

If admin dashboard doesn't show messages:
  1. Verify you're logged in as admin
  2. Check user role in database (should be "admin")
  3. Verify MONGODB_URI is set
  4. Check if messages actually exist in MongoDB

For full troubleshooting, see: TROUBLESHOOTING.md
